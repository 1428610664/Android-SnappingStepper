# Android-SnappingStepper
一种漂亮的UI控件，能更灵活的控制数字的增减。
对应的iOS版 [SnappingStepper](https://github.com/yannickl/SnappingStepper)

####使用gradle 依赖:
```java
compile 'com.bigkoo:snappingstepper:1.0.1'
```
